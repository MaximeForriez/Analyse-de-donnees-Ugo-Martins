## Présentation du projet

Ce dépôt GitHub constitue mon portfolio numérique réalisé dans le cadre du cours d'**Initiation à l'analyse de données avec Python**, dispensé par M. Maxime Forriez à Sorbonne Université.

L'objectif de ce travail est de démontrer l'acquisition de compétences en :
* **Programmation Python** (Traitement de données, boucles, fonctions).
* **Data Science** (Bibliothèques Pandas, Scipy, Matplotlib).
* **Statistiques** (Analyses univariées, bivariées, tests statistiques et régressions).
* **Versionning** (Gestion de projet avec Git/GitHub).

## Structure du Portfolio

L'architecture du dépôt respecte la progression pédagogique du cours :

* **`Seance_04/`** : Distributions statistiques (Lois discrètes et continues).
* **`Seance_05/`** : Statistiques inférentielles (Échantillonnage et Estimation).
* **`Seance_06/`** : Statistique d'ordre (Analyses de rangs, Spearman, Kendall).
* **`Seance_07/`** : Corrélations et Régressions (Analyse PIB vs Énergie).
* **`Seance_08/`** : Analyse bivariée qualitative (Test du Chi2, tables de contingence).
* **`Rapport/`** : Contient le rapport d'analyse final au format PDF.

## Outils utilisés

* **Langage :** Python 3
* **Environnement :** Google Colab
* **Bibliothèques principales :** `pandas`, `numpy`, `matplotlib`, `scipy`.
